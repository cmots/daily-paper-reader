## 摘要
由于高层推理与低层控制的纠缠以及策略优化的不稳定性，视觉-语言-动作模型在实际部署中面临重大挑战。在本文中，我们介绍 SyVLA，一种通过多样化经验训练的鲁棒 VLA 模型。我们提出一种意图解耦算法，从推理上下文中分离出控制相关特征，并利用相似样本引导的强化学习流程来稳定策略更新并减轻分布偏移。在真实机器人任务和多模态基准上的大量实验表明，与现有方法相比，SyVLA 实现了更高的任务成功率和更强的分布外泛化能力，同时有效保留了核心视觉-语言能力。代码和数据集已在项目页面发布。

## Abstract
Vision-Language-Action models face significant challenges in real-world deployment due to the entanglement of high-level reasoning with low-level control, and the instability of policy optimization. In this paper, we introduce SyVLA, a robust VLA model trained with diversified experiences. We propose an Intention Decoupling algorithm to isolate control-relevant features from reasoning contexts and a similar-sample guided RL pipeline to stabilize policy updates and mitigate distribution shift. Extensive experiments on real-world robotic tasks and multi-modal benchmarks demonstrate that SyVLA achieves superior task success rates and stronger out-of-distribution generalization compared to existing methods, while effectively preserving core vision-language capabilities. Codes and Datasets is released on \href{https://sy-vla.github.io/}{project page}.

### 1. 论文的核心问题与整体含义（研究动机和背景）

- **核心问题**：视觉-语言-动作（VLA）模型在现实世界部署中面临两个关键瓶颈：
    1.  **高级推理与低级控制的纠缠**：VLA模型过度关注动作学习，导致其底层视觉-语言模型（VLM）的固有推理和常识理解能力大幅退化，甚至出现灾难性遗忘。现有的“先思考后行动”范式虽然赋予模型推理能力，但高级推理信息会泄露到动作控制中，导致动作不精确或犹豫不决。
    2.  **策略优化的不稳定性**：基于模仿学习的VLA模型，由于训练目标（拟合专家动作）与实际任务成功（长期闭环执行）之间存在鸿沟，容易因误差累积导致分布偏移和任务失败。直接应用强化学习（RL）于十亿参数级别的VLA模型，则极易引发训练不稳定、策略漂移甚至能力崩溃。
- **整体含义**：本文旨在通过多样化经验训练一个鲁棒的VLA模型，解决上述两个核心问题，实现机器人任务能力与视觉-语言理解能力的良好平衡，并提升模型在现实世界中的稳定性和泛化能力。

### 2. 论文提出的方法论：核心思想、关键技术细节

论文提出了名为 **SyVLA** 的模型和一套三阶段训练流程，核心思想是“解耦与稳定”。

- **模型架构：双系统设计**
    - **VLM（视觉-语言模型）**：作为高级感知、语言推理核心和控制意图生成器。采用 Qwen2.5VL-3B。
    - **Action Expert（动作专家）**：基于 Flow Matching 模型，根据控制意图和多模态观测生成精细化的连续动作。
    - **连接机制**：通过一组可学习的 **Feature Query Tokens** 连接VLM和Action Expert。VLM在生成推理的思维链（CoT）后，追加这些Token，其最后一层的隐藏状态（Feature Query States）经过一个MLP适配器后，作为条件输入给Action Expert，指导其生成动作。

- **核心技术一：Intention Decoupling（意图解耦算法）**
    - **问题**：Feature Query States 中混杂了与当前动作无关的高层推理信息，导致Action Expert表现不稳定。
    - **原理**：计算动作损失函数对每个 Feature Query State 的梯度。根据理论分析，梯度L2范数小的Token包含的控制信息较少，主要包含冗余或推理信息。
    - **算法**：
        1.  进行一次前向和反向传播，计算梯度 G。
        2.  计算每个梯度向量的 L2 范数。
        3.  将范数低于某个分位数阈值 τ 的Feature Query State 的特征值置零（掩码），从而抑制冗余信息。
    - **效果**：该方法计算开销小（仅需对Action Expert进行额外一次前向/反向，不更新模型），并能显著提升模型的动作精度。

- **核心技术二：Similar-Sample Guided RL（相似样本引导的强化学习流程）**
    - **问题**：直接在真实机器人上应用RL训练VLA模型，难度高、风险大、易崩溃。
    - **解决方案**：在RL更新时，除了使用从真实环境中采集的样本外，还从专家数据集中检索与当前样本语义相似的样本，共同组成训练批次。其关键设计是：
        - **检索方法**：基于多视角图像编码后的特征向量的余弦相似度进行检索。
        - **优势函数设置**：为了避免价值模型对专家数据的高方差估计导致训练崩溃，人为将所有专家样本的**优势（Advantage）设置为1.0**，从而将学习目标简化为最大化专家动作的似然，而非鉴别“好的专家动作”与“更好的专家动作”，极大稳定了训练过程。
