## 摘要
离散视觉-语言-动作（VLA）模型通常将动作生成公式化为在离散动作空间上的下一个词元预测，每个词元自回归地基于先前上下文进行条件生成。虽然有效，但这种范式会导致高推理延迟，并且很大程度上忽略了动作轨迹中固有的时间结构。最近的工作引入并行解码以提高效率，实现更快的推理，但缺乏对词元依赖关系进行建模的显式机制。我们提出TBD-VLA，一种基于离散词元的VLA框架，它结合了块扩散以实现时间动作生成。我们将动作序列划分为时间块，并在每个块内执行掩码离散扩散，同时保持跨块的自回归生成。这种设计统一了时间自回归和并行动作解码，实现了强时间连贯性和更快的推理速度。此外，显式的时间建模通过时间修补实现了动作块（例如，实时分块）的异步执行。TBD-VLA在仿真和真实操作任务中都显著优于先前的VLA方法，为构建快速、具有时间感知能力的离散VLA模型提供了一条可扩展的路径。项目网页：https://tbd-vla.github.io/

## Abstract
Discrete Vision-Language-Action (VLA) models typically formulate action generation as next-token prediction over discretized action spaces, conditioning each token autoregressively on prior context. While effective, this paradigm incurs high inference latency and largely ignores the temporal structure inherent in action trajectories. Recent efforts introduce parallel decoding to improve efficiency, enabling faster inference, but lack explicit mechanisms for modeling token dependencies. We introduce TBD-VLA, a discrete token-based VLA framework that incorporates block diffusion to enable temporal action generation. We partition action sequences into temporal blocks and perform masked discrete diffusion within each block, while maintaining autoregressive generation across blocks. This design unifies temporal autoregression and parallel action decoding, achieving both strong temporal coherence and improved inference speed. In addition, the explicit temporal modeling enables asynchronous execution of action chunks (e.g., Real-Time Chunking) via temporal in-painting. TBD-VLA significantly outperforms prior VLA approaches in both simulation and real-world manipulation tasks, offering a scalable path toward fast, temporally aware, discrete VLA models. Project webpage: https://tbd-vla.github.io/

- **核心问题**：现有离散视觉-语言-动作（VLA）模型通常将动作生成视为离散动作空间上的下一个词元（token）预测，每个token自回归地基于先前上下文生成。这种范式存在两个主要缺陷：
  - **高推理延迟**：逐token解码速度慢，难以满足闭环、高频机器人控制的需要。
  - **忽视时间结构**：动作轨迹具有固有的时间连贯性，但纯自回归解码缺乏对时间依赖关系的显式建模。
- **研究动机**：近年来虽有并行解码（如OpenVLA-OFT）和离散扩散VLA（如Disc Diff VLA）尝试加速推理，但现有方法要么牺牲时间建模能力，要么缺乏对token依赖关系的显式机制。因此，需要一种既能实现快速并行解码、又能保留时间结构建模的新框架。
- **整体含义**：本文提出TBD-VLA，通过“块扩散（block diffusion）”统一时间自回归与并行解码，在保持强时间连贯性的同时显著提升推理速度，为构建快速、时间感知的离散VLA模型提供可扩展路径。

## 2. 方法论：核心思想、关键技术细节、公式或算法流程

### 核心思想
将动作序列划分为等长的**时间块（temporal block）**，在每个块内执行掩码离散扩散（masked discrete diffusion）并行生成所有token，而在块之间保持自回归生成（前一块的干净token作为下一块的条件）。这一设计结合了块内并行解码的高效性和跨块的时间自回归建模。

### 关键技术细节
1. **动作token化**：将连续动作值离散化为 \(N_b\) 个bin，每个bin对应一个token，动作维度 \(D_a\)，动作块长度 \(H_p\)，总token序列长度 \(L_t = H_p \cdot D_a\)。
2. **时间块划分**：将长度为 \(H_p\) 的动作序列分为 \(K = H_p / m\) 个块，每个块含 \(m\) 个时间步的动作token（实验中使用 \(m=4\)）。
3. **前向加噪过程**：对每个块 \(k\)，独立地对每个token以概率 \(t_{k,i} \sim U(0,1)\) 将其替换为 [MASK] token，其余保持原样。
4. **反向去噪过程**：对于第 \(k\) 个块，模型根据以下条件预测干净token：
   - 对于第一个块（\(k=0\)），条件为锚定块（全[MASK]）和当前噪声块。
   - 对于 \(k>0\)，条件为之前所有块的干净token \(x_{0: k-1}\) 和当前噪声块。
   训练目标是被掩码token上的平均交叉熵损失。
5. **训练中的Token Shift**：将预测目标从自重建改为“当前块预测下一块”，以对齐VLM的next-token prediction预训练目标。
6. **块级注意力掩码（doubled-layout trick）**：将干净序列 \(x_0\) 和噪声序列 \(x_t\) 拼接作为输入，通过自定义注意力掩码实现块内并行、块间自回归，一次前向即可学习所有块的去噪。
7. **推理优化**：
   - **按需解码**：只生成执行所需的动作块（\(K_{exec} = \lceil H_a / m \rceil\)）。
   - **前缀KV缓存**：缓存视觉、文本提示及已生成动作块的KV状态，避免重复计算。
   - **期望采样**：对每个动作维度，从完整预测分布计算期望值作为最终动作值，而非argmax。
   - **实时分块（RTC）**：异步生成未来动作，通过“硬修补”将已执行的动作tail冻结作为后续块的条件，与训练中的掩码扩散对齐。

### 算法流程（文字描述）
1. 输入：观测图像、语言指令、机器人状态。
2. 将状态和指令token化，并插入占位符引导生成L_t个动作token。
3. 将动作序列划分为K个时间块，每个块初始化为全[MASK]。
4. 对于块索引 \(k=0\) 到 \(K-1\)：
   - 若 \(k=0\)，条件为全[MASK]锚定块和当前噪声块；若 \(k>0\)，条件为前 \(k\) 个干净块和当前噪声块。
   - 进行 \(n_d\) 次（实验中 \(n_d=2\)）离散扩散去噪步骤，每次预测所有被掩码位置，只保留高置信度的token。
   - 生成完整干净块并加入KV缓存。
5. 输出所有动作块（或仅所需的 \(K_{exec}\) 个块）。

## 6. 主要结论与发现

1. **SOTA性能**：在LIBERO上平均成功率97.7%，超越所有基线（包括π0.5的96.9%）。在LIBERO-Plus扰动条件下平均83.5%，领先第二方法15.1%。
2. **RTC有效性**：在模拟延迟（L=4）下，TBD-VLA带RTC保持了93.2%的成功率，而w/o RTC降至72.3%，提高20.9个百分点；真实世界中RTC使平均成功率从60.0%提升至67.1%。
3. **推理速度**：通过块扩散设计及推理优化（按需解码、KV缓存、VLM编译），TBD-VLA在LIBERO上延迟仅0.086s（单A40 GPU），与快速方法（如OpenVLA-OFT的0.031s）接近，但时间建模更强。
4. **预训练增益**：预训练显著提升LIBERO-Plus鲁棒性（平均从66.2%升至83.5%），尤其在相机视角（+58.4%）、传感器噪声（+28.3%）和语言指令（+25.2%）扰动上。
5. **期望采样优于argmax**：使用完整分布期望值比argmax高7.1个百分点（88.7% vs 81.6%）。
6. **块大小选择**：m=4实现最佳平衡；m=1（全自回归）延迟高且性能略低；m=16（无时间自回归）性能下降。

## 7. 优点

- **创新性**：首次将块扩散（block diffusion）引入VLA框架，自然结合了并行解码和时间自回归，概念简洁但有效。
- **时间建模能力**：通过块间自回归显式建模动作轨迹的时间结构，而块内并行解码保持高效。
- **推理优化全面**：按需解码、KV缓存、VLM编译、期望采样、RTC等一系列优化，使延迟与快速方法可比。
- **鲁棒性强**：在LIBERO-Plus的7种扰动和真实世界OOD场景下均表现优异，验证了方法的泛化性和抗干扰能力。
- **可扩展性**：与任何VLM骨干兼容，以2B模型达到甚至超越更大模型的性能，说明其高效性。
- **实验设计规范**：覆盖模拟和真实场景，扰动类型丰富，消融实验详实，基线对比广泛，评估标准统一。

## 8. 不足与局限

- **方法层面**：
  - 未探索与辅助VLM目标（如语言建模、视觉问答）的联合训练，可能未能充分利用VLM的语义能力。
  - 块扩散的训练-推理对齐依赖于特定的掩码策略，对噪声调度和步数的选择较敏感（实验表明n_d=2最优，但未对比更多步数）。
- **实验层面**：
  - 真实世界任务仅3个，且每个任务仅50个演示，规模较小，可能无法充分反映复杂操作场景。
  - 真实世界基线仅对比了π0.5，未与更多离散VLA方法（如Disc Diff VLA、OpenVLA）在相同设置下比较。
  - 模型大小差异：TBD-VLA使用2B VLM，而部分基线（如OpenVLA 7B）更大，性能对比可能受参数量影响（尽管作者声称高效）。
- **应用限制**：
  - 在“Transfer Liquid”任务中，修改相机视角导致完全失败（0%成功率），说明对视觉分布变化仍敏感，尤其需要精细视觉保真度的任务。
  - 推理延迟虽低（0.086s），但尚未达到极低延迟（如<10ms），在高频控制（>100Hz）场景下仍有挑战。
  - 离散化bin数量（512）可能限制动作精度，尽管期望采样部分缓解了该问题。
- **其他**：论文未讨论在未见过的机器人形态或大型异构数据集上的零样本迁移能力，预训练数据的多样性仍需验证。
