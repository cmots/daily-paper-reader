## 摘要
组合泛化仍然是深度强化学习（DRL）中的一个核心挑战。经典规划通过显式关系描述提供了一个简单但具有挑战性的研究环境，无需从感知中学习。在稀疏奖励领域中，通过实时搜索进行标准RL探索是无效的，而基于学习的规划方法通常依赖专家演示、事后重新标记或从目标状态开始的随机游走。相比之下，规划器依赖于最佳优先搜索方法（如$\mathrm{A}^\star$）从头开始解决问题。我们提出了一种自我改进的$\mathrm{WA}^\star$学习框架，结合由关系图神经网络表示的价值启发式：启发式引导搜索，搜索产生的数据通过$Q$-学习更新启发式。这一循环产生了可作为通用策略的启发式，甚至无需搜索即可解决新实例，而DRL在其他情况下会失败，正如我们在Sokoban、PushWorld、The Witness以及2023年国际规划竞赛基准等谜题中所展示的。值得注意的是，我们展示了强大的零样本泛化能力：例如，在少于30个方块的Blocksworld实例上训练的启发式，无需搜索即可成功解决包含488个方块的实例。

## Abstract
Combinatorial generalization remains a central challenge in Deep Reinforcement Learning (DRL). Classical planning provides a simple yet challenging setting to study this problem through explicit relational descriptions, without requiring learning from perception. In sparse-reward domains, standard RL exploration via real-time search is ineffective, and learning-based planning methods often rely on expert demonstrations, hindsight relabeling, or random walks from the goal state. In contrast, planners rely on best-first search methods such as $\mathrm{A}^\star$ to solve problems from scratch. We propose a self-improving $\mathrm{WA}^\star$ learning framework in combination with a value heuristic represented by a Relational Graph Neural Network: the heuristic guides search, and the resulting search data updates the heuristic via $Q$-learning. This loop yields heuristics that can function as general policies and solve new instances even without search, where DRL otherwise fails, as we show on puzzles such as Sokoban, PushWorld, The Witness, and the 2023 International Planning Competition benchmarks. Notably, we demonstrate strong zero-shot generalization: For example, heuristics trained on Blocksworld instances with fewer than 30 blocks successfully solve instances with 488 blocks without search.

### 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程（用文字说明）
- **核心思想**：**广义搜索学习（GSP）**——在训练阶段使用**最佳优先搜索（WA*）** 代替RL中常用的实时搜索（agent-centered search），利用当前学习的Q值（来自由关系图神经网络R-GNN表示的价值启发式）引导搜索；搜索产生的轨迹和搜索得出的回报下界作为训练数据，通过**Q-learning**更新Q值，从而形成“搜索指导学习，学习改进搜索”的自增强循环。
- **关键技术细节**：
  - **Q值表示**：使用**关系图神经网络（R-GNN）** 作为Qθ(s,a)的主体。输入是当前状态s和目标g的原子集，加上每个可行动作对应的动作对象和动作原子；通过多层消息传递（每个谓词对应一个组合函数Comb_p，共享更新函数Comb_U）得到每个对象的嵌入；最后用共享MLP将动作对象的嵌入与全局池化特征拼接，输出Q(s,a)。
  - **搜索过程（算法1）**：对每个实例，用**加权A*（WA*，权重w=2）** 在状态-动作节点(s,a)（而非仅状态节点）上进行搜索。搜索维护一个优先队列（按f = g + wQ打分，g是累计回报的下取负数，即负深度）。对每个扩展节点，生成后继状态s'；区分三种情况记录训练数据：
    - **目标路径**：从目标回溯到根，为路径上每个(s,a)分配它所得的实际回报下界R（负步数），存入缓冲区。
    - **死路**：若s'是死胡同（无可用动作），给(s,a)分配固定惩罚R⊥。
    - **非终止**：将(s,a, –∞) 存入缓冲区（表示仅作无约束引导样本）。
  - **Q-learning 目标**：对非终止对使用标准1步贝尔曼目标 ŷ = -1 + max_{a'} Qθ(s',a')；死路对直接回归到R⊥；目标路径对的监督目标为 y = max{R, ŷ}，确保值不因引导而退化。
  - **实例选择策略**：根据问题求解状态（unsolved/solved/satisficed）使用指数加权重采样，优先选择尚不能完美求解的实例，使学习更加高效。
- **使用方式**：测试时，学习到的Q值既可以作为贪婪策略（GSPπ：直接选Q最大的动作）直接执行，也可以作为WA*的启发式（GSPWA*）来引导搜索。

### 6. 论文的主要结论与发现
- **主要贡献**：提出了一个**统一的、自我改进的广义搜索学习框架（GSP）**，通过**最佳优先搜索**替代实时搜索实现高效探索，配合R-GNN学习通用Q值，在无需专家演示或反向随机游走的条件下取得了优异的泛化性能。
- **核心发现**：
  - 学习到的Q值作为**局部动作排序器**（贪婪策略）比作为**全局校准启发式**（WA*引导）更强，因为GNN直接编码了同一状态下所有动作的相对关系，但跨状态的值校准不足。
  - **零样本大规模泛化**是可行的：在Blocksworld上，仅训练≤29块的实例，即可解决488块的实例（贪婪策略覆盖率达100%）。
  - 对于有死路的域（如The Witness），死路监督信号显著提升了策略质量。
  - **AlphaZero**风格的MCTS训练在该设置下完全失败（除Spanner外），证实了MCTS在确定性、稀疏奖励规划中不适合作为主搜索机制。
- **局限**：R-GNN的1-WL表达力限制了其在Rovers（需要计数/路径推理）和Satellite等域的表现；大规模分支因子（如Childsnack）导致搜索失败；24-Puzzle因无法在预算内找到第一个解而完全无法开始学习。

### 7. 优点：方法或实验设计上的亮点
- **方法亮点**：
  - 用**最佳优先搜索（WA*）替代实时搜索**，更高效地探索状态空间，解决了稀疏奖励下的探索瓶颈。
  - **结合搜索得到的回报下界**作为Q-learning目标的额外监督，避免了值函数因引导而退化。

### 8. 不足与局限：包括实验覆盖、偏差风险、应用限制等
- **实验覆盖不足**：
  - 在24-Puzzle上完全失败（0/1000求解），说明方法**不擅长需要极深度搜索且初始启发式完全无效**的领域（无法在预算内找到第一个解以启动学习）。
  - 在Rovers、Satellite上显著弱于LAMA，原作者归因于1-WL表达力不足，实际是否可通过更强大的GNN（如3-WL）克服？未探索。
- **方法局限**：
  - Q值作为全局启发式**校准不佳**，导致WA*在测试时有时表现弱于贪婪策略；作者提出改进方向是改善跨状态值泛化，但未给出方案。
  - **需要已知转移模型**：搜索要求知道转移函数（planning setting），无法直接应用于像素环境（需额外感知学习）。
  - **对超大分支因子敏感**：Childsnack中初始状态有近4700万可行动作，即使有效启发式，WA*的优先队列可能难以管理；这导致GSP覆盖率为41%而Distincter（利用对称性剪枝）达到71%。
  - **训练不稳定**：部分域（如Spanner）训练曲线显示周期性忘记/过拟合，最终checkpoint虽好但过程不鲁棒。
  - 未与**更先进的RL+搜索方法**（如MuZero在规划上的变体）进行对比，仅比较了基础的AlphaZero，结论可能不完全公平。
- **偏差风险**：
  - IPC 2023基准本身可能存在偏向性——其测试实例特意设计为对搜索算法有挑战性（大分支因子、大深度），GSP的WA*变体在此设置下表现劣于贪婪策略，说明该方法**更适合作为策略而非启发式**，这与一般预期（启发式搜索更鲁棒）相反。
  - 所有实验在**同一套超参数**下进行，未探索域特定调优，虽然展示了泛用性，但可能错过更好的性能上限。
- **应用限制**：方法局限于**确定性、完全可观察、基于关系的规划问题**，无法直接应用于存在随机性、部分可观察或连续控制的情景。
