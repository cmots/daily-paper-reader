## 摘要
我们提出了一种3D思维引导的协同训练框架，使得视觉-语言-动作（VLA）模型能够在动作预测过程中隐式地进行3D空间推理。我们的核心见解是，3D几何感知和3D空间推理是可解耦的能力，可以在不同的特征层次注入。在训练过程中，三个紧密耦合的组件主要在潜在空间中协同工作：（1）为了获得几何先验，一个潜在的3D几何感知模块将中间视觉特征与3D基础模型对齐，在不修改VLM骨干架构的情况下获取低级几何线索。（2）作为补充，一个在线3D推理蒸馏模块通过共享的推理锚点令牌缓解提示引发的推理差距。在3D VLM协同训练期间，该锚点作为第一个输出令牌发出，以鲁棒地编码空间先验。在VLA训练期间，它作为插入任务指令和动作指令之间的输入令牌，将高级空间思维从显式的教师推理提示传递到学生动作提示，而无需生成思维链文本。（3）然后，这些解耦的几何和推理特征通过空间增强的动作集成联合起来，将它们作为层次化的空间条件注入到动作查询令牌中，以防止动作捷径。在部署时，我们的方法仅保留其轻量级适配器来执行隐式3D推理，丢弃用于监督的3D基础模型和教师分支。因此，它纯粹在2D图像上运行，无需3D传感器、外部模型或显式文本生成，同时防止预训练VLM的灾难性遗忘，在LIBERO、LIBERO-PLUS、SimplerEnv和真实世界操作任务上实现了最先进的性能。

## Abstract
We propose a 3D-thinking-guided co-training framework that enables vision-language-action (VLA) models to perform 3D spatial reasoning implicitly during action prediction. Our core insight is that 3D geometry perception and 3D spatial reasoning are distinct capabilities that can be disentangled and injected at different feature hierarchies. During training, three tightly coupled components work in concert primarily within the latent space: (1) To gain geometric priors, a latent 3D geometry perception module aligns intermediate visual features with a 3D foundation model, acquiring low-level geometric cues without architectural modifications to the VLM backbone. (2) Complementing this, an online 3D reasoning distillation module mitigates the prompt-induced reasoning gap via a shared reasoning anchor token. During 3D VLM co-training, this anchor is emitted as the first output token to robustly encode spatial priors. During VLA training, it serves as an input token inserted between the task and action instructions, transferring high-level spatial thinking from explicit teacher reasoning prompts to student action prompts without chain-of-thought text generation. (3) These disentangled geometric and reasoning features are then united by a spatially augmented action integration, which jointly injects them into the action-query tokens as hierarchical spatial conditions to prevent action shortcuts. At deployment, our method retains only its lightweight adapters to perform implicit 3D reasoning, discarding the 3D foundation model and the teacher branch used for supervision. Consequently, it operates purely on 2D images without 3D sensors, external models, or explicit text generation while preventing catastrophic forgetting of the pretrained VLM, achieving state-of-the-art performance on LIBERO, LIBERO-PLUS, SimplerEnv, and real-world manipulation tasks.

# 论文总结：3DThinkVLA

## 1. 核心问题与整体含义（研究动机与背景）
现有视觉‑语言‑动作（VLA）模型主要依赖 2D 图像输入，在物理 3D 环境中进行空间推理时存在关键缺陷：
- **2D‑3D 鸿沟**：2D 语义与 3D 空间推理之间缺少直接桥梁。
- **现有方法局限**：显式注入 3D 信息（点云、深度图）的方法依赖外部传感器或需修改 VLM 骨干；隐式对齐 3D 基础模型的方法可能破坏原始的视觉‑语言对齐，且两者均只注入低级几何信息，缺乏高级 3D 空间推理能力（如推断目标位置、相对朝向）。
- **协同训练的隐藏问题**：简单的共训练策略（VLA 数据 + 3D 数据）虽可缓解灾难性遗忘，但会引入“提示诱导的推理差距”——标准动作提示会导致模型绕过已学到的空间先验，退化为动作捷径。

**整体目标**：在仅使用 2D 图像输入且不依赖额外 3D 传感器、外部模型或显式文本生成的前提下，让 VLA 模型在动作预测过程中隐式地进行 3D 空间推理，同时保持预训练 VLM 的语义能力。

## 2. 方法论：核心思想与关键技术细节
### 2.1 核心思想
- **解耦 3D 几何感知与空间推理**：认为两者是可分离的能力，可以在模型的不同特征层次分别注入。
- **协同训练框架**：在 VLA 数据与真实 3D 推理数据（含显式 3D 空间推理问答）上共同训练 VLM 骨干。
- **训练时注入，推理时轻量**：训练时使用三个紧密耦合的组件注入先验；推理时仅保留轻量适配器，丢弃 3D 基础模型和教师分支。

### 2.2 三大关键技术组件
1. **潜在 3D 几何感知模块**  
   - 从视觉编码器第 18 层提取中间视觉特征，通过轻量 **几何适配器**（MLP + LayerNorm）投影到几何潜在空间。
   - 与 3D 基础模型 VGGT 输出的 3D 几何特征进行**块级对齐**（余弦相似度损失 L_geo），获取低级几何线索（深度、表面法向等）。
   - 无需修改 VLM 骨干架构。

2. **在线 3D 推理蒸馏模块**  
   - 设计一个共享的 **推理锚点 token** τ_R，在两类提示中共用：
     - **教师分支**（3D 推理提示）：将 τ_R 作为第一个输出 token 发射，强制编码空间推理先验。
     - **学生分支**（动作提示）：将 τ_R 作为输入 token 插入任务指令与动作指令之间，吸收视觉与任务语义。
   - 通过 **推理适配器** 将学生 τ_R 的隐藏状态投影到推理潜在空间，与教师 τ_R 的隐藏状态进行**令牌级蒸馏**（余弦相似度损失 L_reasoning），将高级空间思维从显式推理提示转移到动作提示路径，无需 CoT 文本生成。
   - 教师与学生共享参数，教师梯度停止，保证训练稳定性。

3. **空间增强动作集成模块**  
   - 将几何适配器的输出（低级几何信息）和推理适配器的输出（高级推理信息）分别投影到动作潜在空间，与动作查询 token 的隐藏状态进行**元素级相加**，形成层次化空间条件输入动作头。
   - 通过随机丢弃部分几何/推理特征来减少过拟合，平衡三者影响。

### 2.3 训练目标与推理路径
- **VLA 阶段损失**： L_vla = L_action + λ_a L_geo + λ_d L_reasoning （L_action 为 L1 距离）
- **VLM 阶段损失**： L_vlm = λ_3D L_CE （标准自回归交叉熵，要求模型将 τ_R 作为第一个输出 token）
- **总损失**： L_total = L_vla + L_vlm，每个训练步内执行一次 VLA 前向和一次 VLM 前向，累积梯度后更新。
- **推理时**：仅保留几何适配器和推理适配器，输入为 2D 图像 + 标准动作提示（含 τ_R），输出动作，无需 3D 传感器、外部模型或显式文本。

## 6. 主要结论与发现
1. **提示诱导的推理差距**是 VLA 共训练的核心障碍：标准动作提示会关闭已学到的 3D 空间先验，退化为动作捷径。
2. **解耦几何感知与空间推理**，并在不同特征层次注入，可以同时获得低级几何线索和高级推理能力，避免恶性相互干扰。
3. **在线潜在自蒸馏**（通过共享推理锚点 token）能够在不生成显式 CoT 文本的前提下，将空间推理从教师提示转移到动作提示，训练稳定且高效。
4. **空间增强动作集成**（联合注入几何与推理特征）有效防止动作捷径，强化空间条件。
5. **整体方法**在 LIBERO（平均 98.7%）、LIBERO‑PLUS（平均 81.0%）、SimplerEnv（平均 72.9%）以及真实任务上均达到当时 SOTA，且推理时无需额外传感器或模型，保持与标准 VLA 相同的接口效率。

## 8. 不足与局限
1. **训练开销增加**：由于双前向（VLA + VLM）和蒸馏计算，训练时间约为标准 VLA 的 1.5 倍，可能限制对更大规模模型的扩展。
