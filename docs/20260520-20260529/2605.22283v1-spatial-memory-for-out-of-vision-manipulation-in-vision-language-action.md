## 摘要
我们提出了SOMA，一种用于视觉-语言-动作（VLA）模型中视野外操作的空间记忆框架。现有的大多数VLA隐式假设任务相关对象始终可见，当目标落在相机视野之外时，会导致脆弱和反应性的行为。SOMA通过为VLA配备一个由可移动头部相机获取的多视角观测构建的持久空间记忆来解决这一限制，从而实现对当前视觉视锥之外的推理。该框架包含三个组件：空间记忆构建，通过扫描将角度方向的观测聚合为统一的空间-语义表示；动态记忆精炼，随时间保持全局一致性；以及上下文记忆检索，在操作过程中激活与指令相关的空间线索。我们在五个具有挑战性的真实世界视野外操作任务上评估了SOMA，包括目标物体初始不可见的多步骤和双臂场景。实验结果表明，SOMA不仅提高了任务成功率，还引发了定性不同的操作行为，包括更快的目标定位、减少视点搜索以及在部分可观测性下近乎一次抓取。在RoboCasa GR1和SimplerEnv上的额外实验进一步验证了SOMA记忆设计在传统完全可观测设置下的有效性。代码即将发布。

## Abstract
We introduce SOMA, the Spatial Memory framework for Out-of-Vision Manipulation in Vision-Language-Action (VLA) models. Most existing VLAs implicitly assume that task-relevant objects are always visible, leading to brittle and reactive behaviors when targets fall outside the camera's field of view. SOMA addresses this limitation by equipping VLAs with a persistent spatial memory constructed from multi-view observations acquired via a movable head camera, enabling reasoning beyond the current visual frustum. The framework consists of three components: Spatial Memory Construction, which aggregates angular-wise observations into a unified spatial-semantic representation through scanning; Dynamic Memory Refinement, which maintains global consistency over time; and Contextual Memory Retrieval, which activates instruction-relevant spatial cues during manipulation. We evaluate SOMA on five challenging real-world out-of-vision manipulation tasks, including multi-step and dual-arm scenarios where target objects are initially invisible. Experimental results show that SOMA not only improves task success rates, but also induces qualitatively different manipulation behaviors, with faster target localization, reduced viewpoint search, and near one-shot grasping under partial observability. Additional experiments on RoboCasa GR1 and SimplerEnv further validate the effectiveness of SOMA's memory design under conventional fully observable settings. Code will be released soon.

# 空间记忆驱动的视野外操作（SOMA）论文总结

## 1. 核心问题与研究动机
- **问题**：现有视觉-语言-动作（VLA）模型隐式假设任务相关对象始终在相机视野内，当目标物体移出视野（Out-of-Vision, OOV）时，模型只能依赖瞬时视觉做出反应，无法利用历史空间信息，导致操作失败。
- **动机**：突破“视图绑定”限制，使机器人能够利用可移动摄像头主动探索环境，构建并持久维护一个全局的空间-语义记忆，实现对当前视野之外物体的推理与操作。

## 2. 方法论
- **核心思想**：为VLA模型配备一个由多视角观测构建的持久空间记忆，通过“构建-精炼-检索”循环支持跨越当前视野的操作。
- **技术细节**：
  - **空间记忆构建**：利用可移动头部摄像头执行预定义扫描策略，获取多帧视频。对采样帧依次使用：
    - 几何先验网络（VGGT）估计相机位姿和粗略3D几何。
    - 语义感知模块（YOLO）检测物体类别和2D边界框。
    - 语义编码器（DINOv3）提取高维视觉特征。
    - 将2D检测通过位姿提升到全局3D坐标系，得到3D边界框。
    - 跨帧进行实例关联（基于外观特征余弦相似度和3D空间一致性），融合为全局实例集合。每个实例表示为三元组（外观嵌入、类别、3D边界框）。
    - 将3D边界框编码为位置向量，与外观嵌入相加得到统一的记忆令牌，集合为初始场景记忆 $M_0$。
  - **动态记忆精炼**：在操作过程中，新的头部观测 $o_t^h$ 经相同管道生成当前记忆令牌 $M_t$。通过类感知实例关联找到匹配的历史记忆，利用语义相似度 $s_{tkj}$ 和动态融合分数 $g_{tkj}$ 计算自适应更新系数 $\alpha_{tkj}$，执行指数移动平均更新：
    \[
    m_t^k = \alpha_{tkj} m_t^j + (1 - \alpha_{tkj}) m_{t-1}^k
    \]
    未匹配的实例追加为新条目，历史条目保留以处理临时遮挡。
  - **上下文记忆检索**：将VLM生成的视觉-语言令牌 $X_{vl}$ 作为查询，精炼后的记忆 $\hat{M}_t$ 作为键-值对，通过交叉注意力增强当前表示：
    \[
    X_{boost} = \text{softmax}\left(\frac{QK^T}{\sqrt{C}}\right)V
    \]
    增强后的特征注入后续DiT块，与当前观测、机器人状态、噪声动作嵌入一起解码出动作序列。

## 6. 主要结论与发现
- SOMA在所有OOV任务中取得最高成功率，且随着任务复杂度增加优势扩大（如双臂协调任务提升最显著）。
- 行为分析显示：SOMA显著缩短首次注视时间（降40–59%）、减少头部搜索路径（降40–57%）、降低视点校正次数和抓取尝试次数，实现近一次性抓取。
- 消融实验证明：空间记忆的几何线索、语义线索、动态更新机制均必不可少，缺失任一部分导致成功率下降4–8%。
- 在完全可观的仿真基准（GR1、SimplerEnv）上，SOMA仍优于或持平现有SOTA，表明记忆设计能提供更稳定的归纳偏置，有利于样本效率和泛化。

## 7. 优点
- **方法创新**：首次将主动扫描+持久空间记忆引入VLA模型，解决视图绑定根本缺陷；模块化设计（构建-精炼-检索）易于集成到现有VLA架构。
- **实验设计**：涵盖部分可观与完全可观场景，提供成功率+行为指标双重评估；消融实验全面（组件、采样间隔、几何来源、推理延迟）。
- **迁移性**：在固定头部相机设置下（无主动扫描）仍表现良好，表明记忆本身具有普遍益处。
- **效率**：离线预处理加速训练，在线推理延迟与基线相当。

## 8. 不足与局限
- **动态鲁棒性**：记忆构建依赖短时扫描和准静态假设，高度动态场景（如移动人员、快速变化）缺乏验证；实例关联阈值需手工设定，可学习性待提升。
- **可扩展性**：当前仅支持桌面级操作，未扩展到移动操作或房间级场景；记忆粒度仅为物体级，无法区分语义状态变化（如抽屉开/关），限制多阶段任务性能。
- **安全与隐私**：持久记忆可能存储错误信息导致物理风险；记忆机制涉及多视角数据，存在隐私和滥用隐患，论文未系统探讨。
- **实验覆盖**：真实世界任务仅基于单一机器人平台（Realman ZM73双臂+自适应夹爪），未在多种形态（如轮式、灵巧手）上验证；仿真任务虽多但均属固定初始视角，未测试长时跨视角记忆漂移。
- **失败模式**：在动态视点下，空间记忆定位准确但仍有约48%的抓取姿态估计错误；完全可观场景中，约32%的失败源于检索到无关记忆，24%源于缺乏任务阶段感知。
