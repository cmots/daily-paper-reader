## 摘要
视觉-语言-动作（VLA）模型在通用机器人操作方面展现出巨大潜力，但由于空间推理不足，尤其是在复杂视觉场景中确定交互位置方面，仍受到限制。尽管近期研究引入了各种形式的视觉规划来解决此问题，但现有方法要么依赖全局几何线索、符号中间表示，要么依赖外部生成的视觉信号，这些通常与下游动作预测的耦合较弱。在这项工作中，我们重新审视VLA系统中的视觉规划，认为有效的规划应当是局部的、视觉可及的、内部生成的，并且直接与动作对齐。基于这一见解，我们提出Afford-VLA，一个统一的框架，将任务条件的可供性内化为VLA模型中的显式视觉规划接口。具体来说，我们引入可学习的<AFF>标记来查询任务相关的交互区域，从多模态特征解码出可供性掩码，并将其转换为紧凑的嵌入，直接条件化动作生成。这种设计使得可供性能够在VLA内部生成并利用，形成紧密耦合的感知-动作通路。为进一步支持这一整合，我们采用一种训练策略，使可供性通路与动作预测联合优化，提高其对下游控制的有效性。我们在多个模拟基准（包括LIBERO、LIBERO-Plus和SimplerEnv）上评估了该方法，取得了持续的最优性能，并在真实世界中获得了强劲结果。这些发现表明，将可供性内化为与动作对齐的视觉规划，为改进VLA系统提供了一种强大的范式。

## Abstract
Vision-language-action (VLA) models have shown strong potential for generalist robot manipulation, yet they remain limited by insufficient spatial reasoning, particularly in determining where to interact in complex visual scenes. While recent efforts introduce various forms of visual planning to address this issue, existing approaches either rely on global geometric cues, symbolic intermediate representations, or externally generated visual signals, which are often weakly coupled with downstream action prediction. In this work, we revisit visual planning in VLA systems and argue that effective planning should be local, visually grounded, internally generated, and directly aligned with action. Based on this insight, we propose Afford-VLA, a unified framework that internalizes task-conditioned affordance as an explicit visual planning interface within VLA models. Concretely, we introduce learnable <AFF> tokens to query task-relevant interaction regions, decode affordance masks from multimodal features, and convert them into compact embeddings that directly condition action generation. This design enables affordance to be both generated and utilized within the VLA, forming a tightly coupled perception-action pathway. To further support this integration, we adopt a training strategy that allows the affordance pathway to be jointly optimized with action prediction, improving its effectiveness for downstream control. We evaluate our method on multiple simulation benchmarks, including LIBERO, LIBERO-Plus, and SimplerEnv, achieving consistent state-of-the-art performance, along with strong real-world results. These findings demonstrate that internalizing affordance as action-aligned visual planning provides a powerful paradigm for improving VLA systems.

## 1. 核心问题与整体含义（研究动机和背景）
- **核心问题**：现有视觉-语言-动作（VLA）模型在通用机器人操作中表现潜力巨大，但在复杂视觉场景中空间推理不足，尤其是难以精准确定“在哪里交互”。这导致操作任务中动作预测缺乏准确的视觉引导。
- **背景**：已有多种视觉规划方法试图增强空间推理，但存在明显局限：
  - **基于几何的方法**（如DepthVLA、4D-VLA）依赖显式或隐式3D线索，提供全局场景级上下文，但缺乏任务相关的局部交互区域聚焦。
  - **基于符号的方法**（如SpatialVLA、RoboPoint）将观察转化为语言或结构化token，间接引导动作，但耦合弱。
  - **视觉可及的方法**（如CoA-VLA、RT-Affordance）通过点、框、掩码等显式视觉线索捕捉交互区域，但往往依赖外部感知模块或作为独立监督目标，与动作预测的耦合度低。
- **论文论点**：有效的视觉规划应满足四个关键属性：**局部性**（聚焦任务相关区域）、**视觉可及性**（直接关联视觉证据）、**内部生成**（由VLA模型自身学习而非外部模块）、**动作对齐**（直接被动作模型消费以影响决策）。目前尚无方法同时满足这些属性。

## 2. 方法论：核心思想、关键技术细节
- **核心思想**：将任务条件的可供性（affordance）内化为VLA模型内部的显式视觉规划接口，形成紧密耦合的感知-动作通路。
- **总体框架**：Afford-VLA以VLA骨干（Qwen3-VL-4B-Instruct）为基础，引入可学习的 `<AFF>` token，通过注意力机制聚合视觉和语言信息，解码出任务条件化的可供性掩码，再将掩码通过掩码池化转化为紧凑的嵌入，直接条件化动作生成。
- **关键步骤**：
  1. **内部可供性掩码生成**：
     - 在指令、图像token序列中插入少量可学习的 `<AFF>` query token。
     - 经过VLM处理后，`<AFF>` token 的隐藏状态条件于当前观测和指令。
     - 通过轻量级解码器（两层双向注意力）结合视觉patch特征，预测逐patch的可供性logits \(G_t \in \mathbb{R}^{H_p \times W_p}\)。
  2. **可供性条件化的动作预测**：
     - 展平 \(G_t\)，选择 top-k logits 对应的 patch 索引，获得二值选择掩码 \(m_t\)。
     - 对被选中的patch特征进行平均池化并投影到VLM隐藏维度，得到可供性嵌入 \(r_t\)。
     - 将 \(r_t\) 附加到VLM隐藏状态序列 \(H_t\) 上，作为动作头（flow-matching DiT-B）的额外条件，预测动作块 \(\hat{a}_{t:t+H}\)。
  3. **动作对齐的训练策略**：
     - 使用**直通梯度估计器（straight-through estimator）**：前向传播为 hard top-k 操作；反向传播中，用softmax代理替代非可微选择，使动作损失梯度能回传到可供性预测头。
     - **两阶段训练**：
       - 第一阶段：仅用密集掩码监督 \(L_{aff}\) 预热可供性头（冻结VLM和动作头）。
       - 第二阶段：模型使用自身预测的logits进行掩码池化，联合优化 \(L_{joint} = L_{act} + L_{aff}\)，实现动作端到端反馈。
  4. **多视图扩展**：对每个视图独立生成可供性掩码和嵌入，将所有视图的嵌入拼接后条件化动作头。

## 3. 实验设计
- **模拟基准（Benchmark）**：
  - **LIBERO**：四个子套件：Spatial、Object、Goal、Long，测试空间推理、物体泛化、目标变化和长程操作。
  - **LIBERO-Plus**：由LIBERO扩展的零样本鲁棒性评测，包含7种扰动：Camera、Robot、Language、Light、Background、Noise、Layout。模型在LIBERO上训练后直接测试。

## 7. 优点
- **方法创新性**：首次提出将affordance作为**内部显式视觉规划接口**，同时满足局部性、视觉可及性、内部生成、动作对齐四个属性，与现有方法（外部affordance、弱耦合）有本质区别。
- **技术细节扎实**：
  - 使用可学习 `<AFF>` token 实现任务条件化，无需额外感知模块。
  - 采用直通梯度估计器实现动作-感知端到端优化，解决非可微池化问题。
  - 两阶段训练策略有效缓解训练-推理不匹配。
- **实验充分且有力**：覆盖主流模拟基准和真实世界任务，进行了多维度消融（集成策略、池化设计、训练策略），所有设计点均得到量化验证。

- **开源承诺**：代码和模型将公开，有利于复现和社区发展。

## 8. 不足与局限
- **依赖外部可供性监督**：当前训练所需的affordance掩码来自外部模型（RAGNet），可能引入偏见和噪声，且限制了完全端到端的自动化。
- **仅2D视觉affordance**：未探索3D几何信息，对于需要深度/点云信息的高精度操作（如堆叠、插入）可能仍有局限。
- **实验场景多样性有限**：虽然涵盖了多个基准，但物体类别、环境类型、机器人形态仍相对有限，向更复杂、非结构化场景的泛化需要进一步验证。
- **动作头为flow-matching特定设计**：可能不能直接迁移到其他动作解码框架（如离散token化、扩散），虽可适配但需额外修改。
- **计算资源需求较高**：使用多块H200 GPU训练，对于资源有限的实验室可能难以复现。
- **真实世界实验任务较少**：仅两个任务，且均为固定工作空间简单操作，大规模真实场景评估不足。
- **潜在安全风险**：affordance预测错误可能导致不安全动作，论文虽提及需安全约束但未深入探讨。
