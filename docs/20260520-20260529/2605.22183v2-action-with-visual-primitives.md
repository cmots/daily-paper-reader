## 摘要
视觉-语言-动作（VLA）模型已成为通用机器人操作的一种有前景的范式。当前架构中的常见设计是将语言指令和视觉观察映射到单次前向传递中的动作。虽然概念上简单，但这种表述将指令理解、空间场景理解和运动控制纠缠在单一学习目标中。因此，动作专家必须隐式地重新学习预训练VLM中已有的认知和感知能力，这可能限制学习效率和泛化能力。我们提出AVP（基于视觉基元的动作），一种端到端架构，实现了这种以视觉基元为中心的接口：VLM推断下一阶段目标并发出视觉基元标记，这些标记条件化一个流匹配动作专家，其监督来自末端执行器运动学。在通用拾取-放置任务上的真实机器人实验表明，AVP相较于pi_0.5将成功率提高了27.61%，并优于其他近期方法，在数据效率、空间组合泛化和物体级迁移方面均取得了一致的提升。

## Abstract
Vision-Language-Action (VLA) models have emerged as a promising paradigm for generalist robotic manipulation. A common design in current architectures maps language instructions and visual observations to actions in a single forward pass. While conceptually simple, this formulation entangles instruction comprehension, spatial scene understanding, and motor control within a single learning objective. As a result, the action expert must implicitly relearn cognitive and perceptual capabilities already present in the pretrained VLM, which can limit both learning efficiency and generalization. We introduce AVP (Action with Visual Primitives), an end-to-end architecture that implements this visual-primitive-centric interface: the VLM infers the next-stage target and emits visual-primitive tokens that condition a flow-matching action expert, with supervision derived from end-effector kinematics. Real-robot experiments on general pick-and-place tasks show that AVP improves the success rate by 27.61% over pi_0.5 and outperforms other recent methods, with consistent gains in data efficiency, spatial-compositional generalization, and object-level transfer.

## 1. 核心问题与整体含义（研究动机与背景）

现有视觉-语言-动作（VLA）模型通常将语言指令理解、视觉场景感知和动作生成耦合在单一前向传递中，导致动作专家必须隐式地重新学习预训练VLM已具备的认知和感知能力。这种耦合限制了学习效率和泛化能力，尤其在空间布局变化或物体替换时表现不佳。尽管已有方法通过规划式分解、子目标图像或视觉提示（如点、框）来解耦，但它们往往依赖外部模型、引入级联误差或增加推理延迟。

本文提出AVP（Action with Visual Primitives），旨在明确划分VLM与动作专家的职责：VLM负责推理“做什么”和“在哪里”，动作专家专注于“如何执行”，两者通过视觉基元（visual primitives）——由VLM内部产生的空间接地标记——进行显式通信。

## 2. 方法论：核心思想、关键技术细节

### 核心思想
- **视觉基元接口**：VLM根据语言指令和观测输出下一阶段目标的视觉基元（如点、包围框、掩码等），这些基元直接投影到图像令牌空间中，作为条件传递给动作专家。
- **端到端训练**：整个架构（VLM、视觉基元解码器、流匹配动作专家）联合优化，无需外部感知模型。

### 关键技术细节
- **架构组件**：预训练VLM → 视觉基元解码器（自回归预测离散化的视觉基元） → 动作专家（流匹配模型，预测动作序列）。

## 6. 主要结论与发现

- AVP在三个任务上的一致改进：中国象棋平均成功率90.28%（π0.5为62.67%），多米诺放置88.19%（π0.5为81.94%），通用物体86.18%（π0.5为64.96%）。

- 视觉基元组件消融：引入框+掩码+记忆取得最佳性能（91%成功率），朴素框也有提升。
- 端到端设计避免了级联延时效影（AVP延迟0.27秒，远低于Point-VLA的37.32秒）。

## 7. 优点

- **明确职责分离**：VLM负责空间推理，动作专家专注于执行，降低学习难度，提升数据效率。
- **可扩展的监督**：视觉基元标签自动从末端运动学生成，无需昂贵的人工标注或外部检测器。
- **端到端优化**：无外部模块，减少误差传播和推理延迟。
- **强泛化能力**：在空间组合和跨域零样本场景均表现出显著优势，说明解耦有助于迁移。
- **硬件友好**：推理单卡RTX 3090即可运行，延迟可接受。
