## 摘要
我们提出了SOMA，即面向视觉-语言-动作（VLA）模型中视外操纵的空间记忆框架。大多数现有VLA模型隐含地假设任务相关对象始终可见，导致当目标超出相机视野时产生脆弱且反应式的行为。SOMA通过为VLA配备一个由可移动头部相机获取的多视角观测构建的持久空间记忆，从而能够超越当前视觉视锥进行推理，解决了这一局限。该框架由三个组件组成：空间记忆构建（通过扫描将角度观测聚合为统一的空间-语义表征）、动态记忆精炼（随时间保持全局一致性）以及上下文记忆检索（在操纵过程中激活与指令相关的空间线索）。我们在五个具有挑战性的真实世界视外操纵任务上评估了SOMA，包括目标对象初始不可见的多步骤和双臂场景。实验结果表明，SOMA不仅提高了任务成功率，还引发了性质不同的操纵行为，包括更快的目标定位、减少的视点搜索以及在部分可观测性下的近单次抓取。在RoboCasa GR1和SimplerEnv上的额外实验进一步验证了SOMA在常规完全可观测环境下的记忆设计的有效性。代码即将发布。

## Abstract
We introduce SOMA, the Spatial Memory framework for Out-of-Vision Manipulation in Vision-Language-Action (VLA) models. Most existing VLAs implicitly assume that task-relevant objects are always visible, leading to brittle and reactive behaviors when targets fall outside the camera's field of view. SOMA addresses this limitation by equipping VLAs with a persistent spatial memory constructed from multi-view observations acquired via a movable head camera, enabling reasoning beyond the current visual frustum. The framework consists of three components: Spatial Memory Construction, which aggregates angular-wise observations into a unified spatial-semantic representation through scanning; Dynamic Memory Refinement, which maintains global consistency over time; and Contextual Memory Retrieval, which activates instruction-relevant spatial cues during manipulation. We evaluate SOMA on five challenging real-world out-of-vision manipulation tasks, including multi-step and dual-arm scenarios where target objects are initially invisible. Experimental results show that SOMA not only improves task success rates, but also induces qualitatively different manipulation behaviors, with faster target localization, reduced viewpoint search, and near one-shot grasping under partial observability. Additional experiments on RoboCasa GR1 and SimplerEnv further validate the effectiveness of SOMA's memory design under conventional fully observable settings. Code will be released soon.

- **整体含义**：论文指出，纯空间推理（如利用大模型推断）不足以弥补感知缺失，需要**持久的地理空间记忆**来存储和复用来自多视角的观测证据。为此提出 **SOMA** 框架，赋予VLA模型在视野外（Out-of-Vision, OOV）操纵能力，同时验证了在全观测场景下记忆设计的有效性。

## 2. 方法论
- **核心思想**：利用可移动头相机主动扫描环境，构建并维护一个统一的空间-语义记忆，在操纵过程中通过动态更新和指令引导的检索来辅助动作生成，实现超越当前视野的推理与操纵。
- **关键技术细节**：
  1. **Spatial Memory Construction（空间记忆构建）**：
     - 在操纵前，若目标未在当前视野中，机器人通过头相机沿预定轨迹扫描获取多视角视频序列。
     - 均匀采样帧，使用VGGT（几何先验网络）估计相机位姿和粗略3D几何，YOLO进行物体检测分类，DINOv3提取视觉特征。
     - 检测到的2D边界框通过VGGT预测的几何信息提升至全局3D坐标，并通过联合外观-几何相似度跨视图关联和融合，得到全局实例集合（外观特征、3D边界框、类别）。
     - 每个实例表示为 `m_k = Φ_mem(f_k) + p_k`，其中 `Φ_mem` 是特征映射，`p_k` 是3D位置嵌入，构成初始记忆 `M0`。
  2. **Dynamic Memory Refinement（动态记忆精炼）**：
     - 在操纵过程中，每时刻的头相机观测 `o_t^h` 经相同流程得到当前记忆 `M_t`。
     - 通过类感知的外观-几何关联找到匹配的记忆条目，计算语义相似度 `st_kj = σ(Φ_sim([m_{t-1}^k - m_t^j]))` 和动态融合分数 `gt_kj = σ(Φ_fuse([m_{t-1}^k, m_t^j]))`，自适应更新系数 `α = g * s`，然后通过指数移动平均更新记忆：`m_t^k = α m_t^j + (1-α) m_{t-1}^k`。
     - 未匹配的新实例作为新条目加入，旧条目保留以处理暂时遮挡。
  3. **Contextual Memory Retrieval（上下文记忆检索）**：
     - VLM产生的视觉-语言token作为查询Q，记忆 `ˆM_t` 经对齐后作为键值(K,V)，通过交叉注意力机制检索相关空间线索，得到增强特征 `X_boost`。
     - 增强特征作为全局上下文注入DiT块，与当前观测、机器人状态、带噪动作拼接，最终由动作解码器输出动作块。

## 6. 主要结论与发现
- SOMA在所有真实OOV任务上成功率最高（平均28.3% vs 其他方法最高约20%），且行为特性显著改善：首次注视时间降低40-59%，头部搜索路径缩短40-57%，抓取尝试接近一次（60% reduction）。
- 消融证实：扫描本身贡献有限，主要是**持久的空间记忆及其动态精炼**带来了性能提升；几何和语义线索均不可或缺。
- 在全观测仿真（RoboCasa GR1）中，SOMA以300演示即达52.0%平均成功率，超过所有基线，且样本效率更高（30演示即优于GR00T-N1.5的Full）。
- 在SimplerEnv上，SOMA在Visual Matching和Variant Aggregation下均达到最佳平均成绩（63.2%和52.5%）。
- **核心发现**：内存引导的操纵范式（memory-guided manipulation）优于纯反应式策略，即使在物体可见时也能提供更强的空间一致性。
